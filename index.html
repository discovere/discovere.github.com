The Tools

The discover-e Community Tools are a collection of programs to process and convert load file formats and perform other common electronic discovery tasks.

The current list of tools is:

  • Customize CTF - A tool to modify Coded Text Files

The Library

The discover-e Common Tools Library is the code library used to build the tools. This code library may be reused and integrated into your custom programming projects.

The library contains a mixture of programming languages, load file format support and other e-discovery domain concepts such as parsing for Bates Numbers.

The current feature set by language is:

  • Python
    • Coded Text File (UTF-8 compliant)
    • Cooncordance DAT (UTF-8 compliant, full-text embedding supported)
  • C#
    • Coded Text File (UTF-8 compliant)
    • Cooncordance DAT (UTF-8 compliant, full-text embedding supported)
    • IPRO LFP
    • Summation DII
    • Bates Numbers
      • Advanced Parsing ofwith prefix and suffix detection
      • Comparison and sorting
      • Range comparison, inclusion and generation
  • Javascript
    • Coded Text File Parser

About

The discover-e Community Tools project provides free tools for the e-discovery community.

It's our opinion that through a spirit of openness and transparency we will grow as a community. We hope these tools will help facilitate that growth by removing barriers to the motion of data.
